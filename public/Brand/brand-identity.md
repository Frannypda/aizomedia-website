# AIZOMEDIA — Identidad Visual

## Paleta de colores

| Nombre | Hex | Uso |
|---|---|---|
| Navy primario | `#0a1a35` | Color principal de marca — fondos oscuros, headers, elementos de peso |
| Navy secundario | `#183059` | Variante más clara del navy — gradientes, hover states |
| Gris medio | `#555555` | Texto secundario, iconos, elementos neutros |
| Gris claro | `#e6e6e6` | Bordes, fondos secundarios, separadores |
| Negro | `#000000` | Texto sobre fondo claro máximo contraste |
| Blanco | `#ffffff` | Fondo base principal de todos los documentos |

## Uso de colores

- **Fondo base:** blanco `#ffffff`
- **Color principal de marca:** navy oscuro `#0a1a35`
- **Gradiente de marca:** `#0a1a35` → `#183059` (de oscuro a medio)
- **Nota:** los gradientes son un elemento clave de la identidad visual

## Logotipo

- **Tipo:** wordmark — "Aizomedia." con punto final
- **Estilo:** bold sans-serif, "A" con corte angular geométrico en la parte izquierda
- **Color primario:** navy `#0a1a35` sobre fondo blanco
- **Archivo:** `assets/logo-aizomedia.png` (guardar aquí el PNG original)
- **Versiones a mantener:** logo color navy, logo blanco (para fondos oscuros)

## Tipografía

- Sistema: -apple-system, BlinkMacSystemFont, "Segoe UI" (a confirmar con tipografía propia)

## Aplicación en documentos

- Headers de auditoría: fondo blanco + barra de acento con gradiente navy
- Secciones oscuras (Sprint Teaser): fondo `#0a1a35` o `#0a0a0a`
- CTAs: botón negro `#000000` o navy `#0a1a35`
- Badges / tags: navy con texto blanco
